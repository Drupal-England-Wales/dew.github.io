We will be using an install of the open source software Helios to vote for the Board, candidates profiles will be posted soon, and the call for Board candidates remains open, though there is currently no program for the initial election itself. We will update this page when that changes.

The originally intended election program remains minuted here.

When elections take place, the method for voting and counting will be “instant run-off voting”. The outright winner will be appointed Chairperson and the next six successful candidates will be appointed Board members, role to be determined at the next meeting of the Board.
